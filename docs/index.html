    • Complex Graphing Calculator

    This is a graphing calculator which lets the user enter a function which can return complex values.

    The calculator can be found here:

    https://oscarlitorell.github.io/complex-graphing/

    The repository can be found here:

    https://github.com/OscarLitorell/complex-graphing

    Usage

    How to use it is documented in the user-manual folder. There is an English version and a Swedish version.

    Documentation

    Documentation can be found here:

    https://oscarlitorell.github.io/complex-graphing/docs/
